# NoHeavyGear.cs
Blocks players from defined piece(s) of clothing from mounting into a vehicle.  
Plugin blocks mounting a vehicle(s) while wearing .JSON defined clothing

Clothing/armor/vehicle types are definded in the .JSON


- **Permission:** *NoHeavyGear.bypass*

This bypasses any restrictons set in the .JSON file*
